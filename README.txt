MIDI Trigger is a project that demonstrates how MIDI inputs can be used to trigger events. Although the events in version 1.0 of MIDI Trigger simply displays a MessageBox, this could be extended to open doors in games or  launch programs on the user's PC.

Thank you for trying MIDI Trigger. You can contact the author, keyboardP, via www.keyboard.me.

MIDI Trigger uses NAudio (http://naudio.codeplex.com). Special thanks to Mark Heath and others who work on the NAudio project.
